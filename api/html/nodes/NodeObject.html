NodeObject¶

class NodeGraphQt.NodeObject(qgraphics_item=None)¶

The NodeGraphQt.NodeObject class is the main base class that all nodes inherit from.

Inherited by: NodeGraphQt.BaseNode, NodeGraphQt.BackdropNode
