Widgets¶

Custom widgets included in the NodeGraphQt module.

Properties Bin¶

The NodeGraphQt.PropertiesBinWidget is a list widget for displaying and editing a nodes properties.

example

1
2
3
4
5
6
7
8
from NodeGraphQt import NodeGraph, PropertiesBinWidget

# create node graph.
graph = NodeGraph()

# create properties bin widget.
properties_bin = PropertiesBinWidget(parent=None, node_graph=graph)
properties_bin.show()

Nodes Tree¶

The NodeGraphQt.NodeTreeWidget is a widget for displaying all registered nodes from the node graph with this widget a user can create nodes by dragging and dropping.

example

1
2
3
4
5
6
7
8
from NodeGraphQt import NodeGraph, NodeTreeWidget

# create node graph.
graph = NodeGraph()

# create node tree widget.
nodes_tree = NodeTreeWidget(parent=None, node_graph=graph)
nodes_tree.show()
class NodeGraphQt.NodeTreeWidget(parent=None, node_graph=None)¶

Node tree for displaying node types.

Parameters

  • parent (QtWidgets.QWidget) – parent of the new widget.

  • node_graph (NodeGraphQt.NodeGraph) – node graph.

set_category_label(category, label)¶

Set custom display label for a node category.

Parameters

  • label (str) – custom display label.
