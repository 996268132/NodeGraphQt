Menus¶

Setup Context Menu¶

The setup_context_menu is a convenience function that’ll populate the main context menu on node graph.

NodeGraphQt.setup_context_menu(graph)¶

populate the specified graph’s context menu with essential menus commands.

example code:

1
2
3
4
from NodeGraphQt import NodeGraph, setup_context_menu

graph = NodeGraph()
setup_context_menu(graph)

result:

Parameters

graph (NodeGraphQt.NodeGraph) – node graph.

Graph Menu¶

The context menu triggered from the node graph.

class NodeGraphQt.NodeGraphMenu(graph, qmenu)¶

The NodeGraphMenu is the main context menu triggered from the node graph.

example for accessing the node graph context menu.

1
2
3
4
5
6
from NodeGraphQt import NodeGraph

node_graph = NodeGraph()

# get the context menu for the node graph.
context_menu = node_graph.get_context_menu('graph')
