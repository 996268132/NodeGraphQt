Builtin Widgets¶

Here’s a few custom widgets that included with the NodeGraphQt framework.

Properties Bin¶

The NodeGraphQt.PropertiesBinWidget is a list widget for displaying and editing a nodes properties.

example

1
2
3
4
5
6
7
8
from NodeGraphQt import NodeGraph, PropertiesBinWidget

# create node graph.
graph = NodeGraph()

# create properties bin widget.
properties_bin = PropertiesBinWidget(parent=None, node_graph=graph)
properties_bin.show()
class NodeGraphQt.PropertiesBinWidget(parent=None, node_graph=None)¶

Node properties bin for displaying properties.

Parameters

  • parent (QtWidgets.QWidget) – parent of the new widget.

  • node_graph (NodeGraphQt.NodeGraph) – node graph.

add_node(node)¶

Add node to the properties bin.

Parameters

node (NodeGraphQt.NodeObject) – node object.

clear_bin()¶

Clear the properties bin.

limit()¶

Returns the limit for how many nodes can be loaded into the bin.

Returns

node limit.

Return type

int

lock_bin()¶

Lock/UnLock the properties bin.

prop_widget(node)¶

Returns the node property widget.

Parameters

node (str or NodeGraphQt.NodeObject) – node id or node object.

Returns

node property widget.

Return type

NodePropWidget

remove_node(node)¶

Remove node from the properties bin.

Parameters

node (str or NodeGraphQt.BaseNode) – node id or node object.

set_limit(limit)¶

Set limit of nodes to display.

Parameters

limit (int) – node limit.

Nodes Tree¶

The NodeGraphQt.NodeTreeWidget is a widget for displaying all registered nodes from the node graph with this widget a user can create nodes by dragging and dropping.

example

1
2
3
4
5
6
7
8
from NodeGraphQt import NodeGraph, NodeTreeWidget

# create node graph.
graph = NodeGraph()

# create node tree widget.
nodes_tree = NodeTreeWidget(parent=None, node_graph=graph)
nodes_tree.show()
class NodeGraphQt.NodeTreeWidget(parent=None, node_graph=None)¶

Node tree for displaying node types.

Parameters

  • parent (QtWidgets.QWidget) – parent of the new widget.

  • node_graph (NodeGraphQt.NodeGraph) – node graph.

set_category_label(category, label)¶

Set custom display label for a node category.

Parameters

  • category (str) – node identifier category eg. “nodeGraphQt.nodes”

  • label (str) – custom display label.

update()¶

Update and refresh the node tree widget.

Publish Node¶

The NodeGraphQt.NodePublishWidget is a simple widget for publishing nodes to a serialized json formatted file.

class NodeGraphQt.NodePublishWidget(node)¶

Node publish dialog widget for publishing nodes.

Parameters

node (NodeGraphQt.BaseNode) – node object.
