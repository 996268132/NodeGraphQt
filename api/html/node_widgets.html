Node Widgets¶

Node widgets are widgets that are embedded in a NodeGraphQt.BaseNode object displayed in the node graph.

NodeBaseWidget¶

class NodeGraphQt.widgets.node_widgets.NodeBaseWidget(parent=None, name='widget', label='')¶

NodeBaseWidget is the main base class for all node widgets that is embedded in a NodeGraphQt.BaseNode object.
